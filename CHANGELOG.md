### Added

- Terraform module provisioning a production-ready SonarQube Community Build on a single EC2 instance: Graviton by default, IMDSv2 required, encrypted volumes, no SSH (SSM Session Manager only).
- Persistent data EBS volume mounted at /var/lib/docker so all state survives instance replacement.
- Database password generated by Terraform and delivered through SSM Parameter Store, rendered on the instance at boot.
- Daily pg_dump backups to S3 and daily EBS snapshots via Data Lifecycle Manager.
- Optional Route53 record, CloudWatch alarms and automatic instance recovery.
- Compose-only deployment path under docker/ for non-AWS hosts.
- Examples: complete (creates a minimal VPC) and existing-vpc.
- Reference backup and restore scripts, GitHub Actions scanner example, full documentation set.
